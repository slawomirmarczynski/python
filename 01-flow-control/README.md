# Programy przykładowe w Pythonie

## heron

Python jako kalkulator.

Podstawy programowania na przykładzie prostych obliczeń (obliczanie pola powierzchni trójkąta),
definiowanie funkcji i przekazywanie parametrów, sposoby dokumentacji kodu.

## loop

Iteracja - tworzenie pętli powtarzanych instrukcji.

## recursion

Rekurencja.

Rekurencja zamiast pętli for - przykład że nawet nie mając ani instukcji goto,
ani pętli for, while itp. - możemy powtarzać określoną liczbę razy wykonanie
wybranego fragmentu programu.

## nan

Nie-liczby i efekty ich porównywania.

Porównując NaN z czymkolwiek, nawet z NaN, zawsze otrzymuje się jako wynik
fałsz (czyli False). Może to powodować zupełnie nieoczekiwane przez mniej
doświadczonych programistów działanie programów.
